# Netflix EDA Project

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Netflix Movies and TV Shows dataset.

## Objectives
• Analyze most popular genres  
• Understand movie vs TV show distribution  
• Study release trends over years  
• Identify top contributing countries

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
• Drama is the most common genre
• Most Netflix content was released after 2015
• Movies dominate the platform compared to TV Shows

## Dataset
Netflix Movies and TV Shows dataset
